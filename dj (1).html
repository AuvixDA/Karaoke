⚙️ Настройка Firebase

Для работы в реальном времени нужно вставить данные вашего Firebase-проекта в оба файла (guest.html и dj.html).

apiKey: "..."
authDomain: "....firebaseapp.com"
databaseURL: "https://...-rtdb.firebaseio.com"
projectId: "..."

Как получить: console.firebase.google.com → Создать проект → Realtime Database → Настройки → Добавить приложение (Web) → скопировать конфиг.
